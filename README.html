#FreelanceApp

We created this app as a school project. It’s our first time that we used AngularJS, MongoDB and Sails

##Installation Manual Sails Backend

  • Open your terminal
  • Change the directory to “backend” folder from our app
  • Run ` npm install
  • Start MongoDB: mongod --config /usr/local/etc/mongod.conf
  • Open new terminal tab and start the API: node app.js
